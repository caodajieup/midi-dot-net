Scale.Traverse Method

Returns the sequence of notes generated by this scale when moving from start to finish.

Parameters

The first note in the traversal.

The last note in the traversal.

Return Value

The sequence of notes. The result always includes start and finish, and then includes whichever intervening notes are implied by scale when moving in that direction.
